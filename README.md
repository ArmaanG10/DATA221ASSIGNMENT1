# DATA221ASSIGNMENT1
Repository containing all eight codes for the corresponding eight questions in the first assignment.
assignment1q1.py runs a loop that multiplies numbers until it hits a certain limit
assignment1q2.py takes some words and makes a dictionary with their lengths
assignment1q3.py does math with exponents but skips the negative ones
assignment1q4.py sorts a list of numbers and finds where they get bigger than a random value
assignment1q5.py asks the user for two circle sizes and says how much they overlap
assignment1q6.py looks at a list of numbers to see the distribution of values
assignment1q7.py asks the user for a big number of seconds and turns it into normal clock time
assignment1q8.py uses pandas to make a small table and add a new column to it.
you will need python 3 and the pandas library installed to run everything.
